# GongJang (공장)

### 공유 기반 장바구니 서비스

**Gong**유하는 **Jang**바구니
_따로 또 같이, 스마트한 장보기의 시작_

<br/>

**"장바구니, 이제 실시간으로 공유하세요"**

GongJang은 실시간 동기화를 지원하는 공유 장바구니 플랫폼입니다.<br/>
함께 장바구니를 실시간으로 공유하고 중복 구매를 방지하세요.

### 솔루션

**GongJang**은 실시간 동기화를 지원하는 공유 장바구니 플랫폼으로, 그룹 구성원 간의 쇼핑 경험을 매끄럽게 연결합니다.

- **실시간 공유** - 장바구니 상태의 즉각적인 동기화
- **동시성 제어** - 다수가 동시에 수정해도 충돌 없는 데이터 무결성 보장
- **효율적 관리** - Client/Server 상태 분리를 통한 최적화된 UX
